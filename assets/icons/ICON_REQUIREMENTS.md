# PWA Icon Requirements

Your PWA needs the following icon sizes for optimal compatibility across all devices and platforms:

## Required Icon Sizes:
- **72x72** - Android small icon
- **96x96** - Android medium icon  
- **128x128** - Chrome Web Store icon
- **144x144** - Windows tile icon
- **152x152** - iOS icon
- **192x192** - Android large icon (maskable)
- **384x384** - Android extra large icon
- **512x512** - Splash screen icon (maskable)

## Icon Guidelines:
1. **Format**: PNG format recommended
2. **Background**: Use solid background color (#0f172a to match your theme)
3. **Content**: Simple, recognizable symbol representing your budget app
4. **Maskable Icons**: For 192x192 and 512x512, ensure the important content is within the safe area (center 80% of the icon)

## Suggested Icon Design:
- A calculator or coin symbol
- The letter "D" for Dhanika
- A chart/graph symbol
- Dollar sign or currency symbol

## How to Generate Icons:
1. Create a base 512x512 icon design
2. Use online tools like:
   - https://realfavicongenerator.net/
   - https://maskable.app/ (for maskable icons)
   - Or any image editor to resize

## File Names Expected:
- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

Once you add these icons to the `/assets/icons/` directory, your PWA will be fully functional!
